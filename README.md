# Positivus — Landing Page Practice

## О проекте
Учебный проект по верстке адаптивного лендинга для агентства цифрового маркетинга. В процессе работы я закреплял навыки построения сложных сеток и работы с семантической разметкой.

## 🛠 Стек технологий
* **HTML5** (семантика)
* **CSS3**
* **CSS Grid** (основная сетка проекта)
* **Flexbox** (позиционирование элементов внутри блоков)
